# 子供向けお絵描きアプリ 🎨

子供が直感的に楽しく使えることを目指したWebアプリです。  
鉛筆、消しゴム、塗りつぶし、保存機能を実装し、遊びながら創造力を育むことを目的としています。

---

## 特徴
- **直感的なUI**  
  絵文字ややさしい言葉を使用し、子供でも迷わず操作できるようにしました。  

- **世界観の工夫**  
  明るくファンタジーなデザインを採用し、楽しさを演出。  

- **基本機能の実装**  
  - 鉛筆ツール：自由に描画  
  - 消しゴムツール：部分消去  
  - バケツツール：クリックした範囲を塗りつぶし  
  - 保存機能：キャンバスを画像として保存  

---

## 技術要素
- 使用技術: JavaScript, HTML, CSS  
- Canvas API を利用した描画処理  
- Vite を利用した開発環境
